# Imágenes de Proyectos

Coloca aquí las imágenes de tus proyectos con los siguientes nombres:

- `project1.jpg` - Proyecto 1
- `project2.jpg` - Proyecto 2
- `project3.jpg` - Proyecto 3
- `project4.jpg` - Proyecto 4
- `project5.jpg` - Proyecto 5
- `project6.jpg` - Proyecto 6

## Recomendaciones:
- Formato: JPG o PNG
- Tamaño recomendado: 800x600px
- Peso máximo: 500KB por imagen
- Relación de aspecto: 4:3 o 16:9
